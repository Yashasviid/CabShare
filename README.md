# CabShare

A full-stack ride-sharing web app built with React, Node.js, Express, and MongoDB.

## Features
- OTP-verified ride booking with real-time driver-passenger matching
- Live route map, in-app chat, and emergency assistance
- Role-based dashboards for drivers and passengers
- Post-ride ratings, ride history, and women-only ride filter

## Tech Stack
React · Node.js · Express · MongoDB · Leaflet Maps

## Setup
```bash
npm install && npm start
```

> Built for safe, affordable intercity carpooling across India.
